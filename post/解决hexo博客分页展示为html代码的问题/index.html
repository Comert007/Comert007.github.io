标签: hexo  

Hexo更新之后，首页的分页展示为HTML代码

打开主题文件themes/你的主题/layout/_partial/pagination.ejs文件，修改代码： 修改前：

<%- paginator({
    prev_text: "<i class='fa fa-angle-left'></i>",
    next_text: "<i class='fa fa-angle-right'></i>"
}) %>

修改后：

<%- paginator({
    prev_text: "<i class='fa fa-angle-left'></i>",
    next_text: "<i class='fa fa-angle-right'></i>",
    escape: false
}) %>
